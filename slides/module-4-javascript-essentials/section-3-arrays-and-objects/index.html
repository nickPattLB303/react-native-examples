Arrays and Objects

Learning Objectives

  • Use object destructuring to extract values efficiently
  • Apply the spread operator for arrays and objects
  • Implement array transformation methods like map, filter, and reduce

  • Understand immutability concepts for React Native state management

Arrays in JavaScript

Arrays are ordered collections of values that are fundamental to React Native development:

  • Used to store lists of data (medications, patients, etc.)
  • Zero-based indexing
  • Can contain mixed types (though not recommended)
  • Dynamic length (can grow and shrink)
// Creating arrays
const medications = ["Aspirin", "Lisinopril", "Metformin"];
const emptyArray = [];
const mixedArray = [1, "text", true, null, {name: "John"}];

// Array constructor (less common)
const newArray = new Array(3); // Array with 3 empty slots
const nameArray = Array.of("Alice", "Bob", "Charlie");

// Array from iterable
const chars = Array.from("ABCDE"); // ["A", "B", "C", "D", "E"]
const numbers = Array.from({length: 5}, (_, i) => i + 1);
// [1, 2, 3, 4, 5]

Array Access and Modification

Common operations:

  • Accessing elements with square brackets
  • Getting array length via length property
  • Setting values with assignment
  • Adding/removing elements with methods
const medications = ["Aspirin", "Lisinopril", "Metformin"];

// Accessing elements (zero-based indexing)
console.log(medications[0]); // "Aspirin"
console.log(medications[2]); // "Metformin"
console.log(medications[3]); // undefined (out of bounds)

// Array length
console.log(medications.length); // 3

// Modifying elements
medications[1] = "Ibuprofen";
console.log(medications); // ["Aspirin", "Ibuprofen", "Metformin"]

// Adding elements (modifies original array)
medications.push("Simvastatin"); // Add to end
medications.unshift("Atorvastatin"); // Add to beginning

// Removing elements (modifies original array)
const last = medications.pop(); // Remove from end
const first = medications.shift(); // Remove from beginning

Array Methods: Adding and Removing

const prescriptions = ["Aspirin", "Lisinopril"];

// Adding elements
prescriptions.push("Metformin");       // Adds to the end
console.log(prescriptions);            // ["Aspirin", "Lisinopril", "Metformin"]

prescriptions.unshift("Simvastatin");  // Adds to the beginning
console.log(prescriptions);            // ["Simvastatin", "Aspirin", "Lisinopril", "Metformin"]

// Removing elements
const last = prescriptions.pop();      // Removes from the end
console.log(last);                     // "Metformin"
console.log(prescriptions);            // ["Simvastatin", "Aspirin", "Lisinopril"]

const first = prescriptions.shift();   // Removes from the beginning
console.log(first);                    // "Simvastatin"
console.log(prescriptions);            // ["Aspirin", "Lisinopril"]

// splice() - insert or remove at specific positions
prescriptions.splice(1, 0, "Amlodipine"); // Insert at index 1, delete 0 elements
console.log(prescriptions);            // ["Aspirin", "Amlodipine", "Lisinopril"]

prescriptions.splice(0, 1);            // Remove 1 element starting at index 0
console.log(prescriptions);            // ["Amlodipine", "Lisinopril"]

React Native Tip: Methods like push, pop, shift, unshift, and splice mutate the original array. When working with state in React Native, use non-mutating approaches to maintain immutability.

Array Methods: Finding Elements

const medications = [
  { id: 1, name: "Aspirin", type: "pain" },
  { id: 2, name: "Amoxicillin", type: "antibiotic" },
  { id: 3, name: "Lisinopril", type: "blood pressure" }
];

// indexOf - find position of simple value (uses strict equality)
const simpleArray = ["Aspirin", "Lisinopril", "Metformin"];
console.log(simpleArray.indexOf("Lisinopril")); // 1
console.log(simpleArray.indexOf("Unknown")); // -1

// find - returns first matching element
const pain = medications.find(med => med.type === "pain");
console.log(pain); // { id: 1, name: "Aspirin", type: "pain" }

// findIndex - returns index of first matching element
const index = medications.findIndex(med => med.name === "Lisinopril");
console.log(index); // 2

// some - checks if at least one element passes test
const hasAntibiotic = medications.some(med => med.type === "antibiotic");
console.log(hasAntibiotic); // true

// every - checks if all elements pass test
const allPainMeds = medications.every(med => med.type === "pain");
console.log(allPainMeds); // false

// includes - checks if array contains a value (simple values only)
console.log(simpleArray.includes("Metformin")); // true

Array Transformation: map()

map() creates a new array by transforming each element:

  • Calls a function on each element
  • Creates a new array with the results
  • Original array remains unchanged
  • One-to-one transformation (same length)

React Native Tip: map() is extensively used to render lists of components from arrays of data.

const medications = [
  { id: 1, name: "Aspirin", type: "pain", dosage: "325mg" },
  { id: 2, name: "Lisinopril", type: "blood pressure", dosage: "10mg" },
  { id: 3, name: "Metformin", type: "diabetes", dosage: "500mg" }
];

// Extract just the names
const medicationNames = medications.map(med => med.name);
console.log(medicationNames); 
// ["Aspirin", "Lisinopril", "Metformin"]

// More complex transformation
const medicationItems = medications.map(med => ({
  label: med.name,
  description: `${med.dosage} (${med.type})`,
  key: med.id.toString()
}));

// In React Native JSX
function renderMedicationList() {

Array Transformation: filter()

filter() creates a new array with elements that pass a test:

  • Tests each element with a function
  • Includes elements that return true
  • Original array remains unchanged
  • Result may be shorter than original
const medications = [
  { id: 1, name: "Aspirin", type: "pain", dosage: "325mg" },
  { id: 2, name: "Lisinopril", type: "blood pressure", dosage: "10mg" },
  { id: 3, name: "Metformin", type: "diabetes", dosage: "500mg" }
];

// Filter by medication type
const painMeds = medications.filter(med => med.type === "pain");
console.log(painMeds.length); // 1
console.log(painMeds[0].name); // "Aspirin"

// Filter by multiple criteria
const highDoseMeds = medications.filter(med => {
  const dosageNumber = parseInt(med.dosage);
  return dosageNumber >= 500;
});
console.log(highDoseMeds.length); // 1

// Combining filter and map
const highDoseNames = medications
  .filter(med => parseInt(med.dosage) >= 100)
  .map(med => med.name);
console.log(highDoseNames); // ["Aspirin", "Metformin"]

Array Transformation: reduce()

reduce() accumulates array values into a single result:

  • Executes a reducer function on each element
  • Accumulates result into a single value
  • Accumulator can be any type (number, string, array, object)
  • Very versatile and powerful
const medications = [
  { id: 1, name: "Aspirin", type: "pain", dosage: "325mg" },
  { id: 2, name: "Lisinopril", type: "blood pressure", dosage: "10mg" },
  { id: 3, name: "Metformin", type: "diabetes", dosage: "500mg" }
];

// Sum numeric values
const pillCounts = [30, 60, 90, 120];
const totalPills = pillCounts.reduce(
  (sum, count) => sum + count, 
  0 // initial value
);
console.log(totalPills); // 300

// Collect unique medication types
const medicationTypes = medications.reduce((types, med) => {
  if (!types.includes(med.type)) {
    types.push(med.type);
  }
  return types;
}, []);
console.log(medicationTypes); 
// ["pain", "blood pressure", "diabetes"]

// Group medications by type
const groupedByType = medications.reduce((groups, med) => {
  const type = med.type;
  if (!groups[type]) {
    groups[type] = [];
  }
  groups[type].push(med);
  return groups;
}, {});

console.log(groupedByType.pain.length); // 1

Array Transformation: sort()

sort() arranges array elements in place:

  • Sorts elements and changes the original array
  • Default sorts as strings (lexicographically)
  • Accepts compare function for custom sorting

Warning: sort() mutates the original array. Always create a copy of your array before sorting in React Native.

const medications = [
  { id: 1, name: "Aspirin", type: "pain", dosage: "325mg" },
  { id: 2, name: "Lisinopril", type: "blood pressure", dosage: "10mg" },
  { id: 3, name: "Metformin", type: "diabetes", dosage: "500mg" }
];

// Default string sort
const fruits = ["orange", "apple", "banana"];
fruits.sort();
console.log(fruits); // ["apple", "banana", "orange"]

// Sorting numbers (need compare function)
const doses = [10, 5, 25, 15];
doses.sort((a, b) => a - b); // Ascending
console.log(doses); // [5, 10, 15, 25]

// Immutable sort with objects
const sortedMeds = [...medications].sort((a, b) => 
  a.name.localeCompare(b.name)
);

// In React Native
function MedicationList({ medications }) {
  const sortedMeds = [...medications].sort(
    (a, b) => a.name.localeCompare(b.name)
  );

Array Spread Operator

The spread operator (...) expands array elements:

  • Copy arrays
  • Combine arrays
  • Create new arrays with additional elements
  • Extract array elements
  • Essential for immutable state updates
const morningMeds = ["Lisinopril", "Vitamin D"];
const eveningMeds = ["Aspirin", "Simvastatin"];

// Combine arrays
const allMeds = [...morningMeds, ...eveningMeds];
console.log(allMeds); 
// ["Lisinopril", "Vitamin D", "Aspirin", "Simvastatin"]

// Create a copy of an array
const medsCopy = [...morningMeds];

// Add element without mutating original
const updatedMorningMeds = [...morningMeds, "Fish Oil"];
console.log(updatedMorningMeds); 
// ["Lisinopril", "Vitamin D", "Fish Oil"]
console.log(morningMeds); 
// ["Lisinopril", "Vitamin D"] (unchanged)

// Insert in the middle
const withInsert = [
  ...allMeds.slice(0, 2),
  "Metformin",
  ...allMeds.slice(2)
];

Objects in JavaScript

Objects are collections of key-value pairs:

  • Store related data and functionality
  • Keys are strings or symbols
  • Values can be any JavaScript type
  • Dynamic (can add/remove properties)
  • Essential for modeling data in React Native
// Object literal
const medication = {
  id: 1,
  name: "Aspirin",

  frequency: "once daily",
  instructions: "Take with food"
};

// Empty object
const emptyObject = {};

// Nested objects
const prescription = {
  patient: {
    id: "P123",

  },

Object: Accessing Properties

const medication = {
  id: 1,
  name: "Aspirin",
  dosage: "81mg",
  frequency: "once daily",
  instructions: "Take with food"
};

// Dot notation
console.log(medication.name); // "Aspirin"
console.log(medication.dosage); // "81mg"

// Bracket notation (useful for dynamic property names)
console.log(medication["instructions"]); // "Take with food"

const propName = "frequency";
console.log(medication[propName]); // "once daily"

// Adding new properties
medication.refills = 3;
medication["prescribedBy"] = "Dr. Smith";

// Modifying properties
medication.dosage = "325mg";

// Deleting properties
delete medication.refills;

// Checking if a property exists
console.log("instructions" in medication); // true
console.log(medication.hasOwnProperty("price")); // false
console.log(medication.nonExistentProp); // undefined

Object Methods

Object methods are functions stored as object properties:

  • Traditional function syntax (creates this context)
  • ES6 shorthand method syntax
  • Arrow functions (inherit this from surroundings)
const patient = {
  firstName: "John",
  lastName: "Doe",
  age: 45,
  medications: ["Aspirin", "Lisinopril"],
  
  // Traditional method (with own 'this')
  getFullName: function() {
    return `${this.firstName} ${this.lastName}`;
  },
  
  // ES6 method shorthand
  getAge() {
    return this.age;
  },
  
  // Arrow function (warning: 'this' isn't bound)
  getMedications: () => {
    // 'this' doesn't refer to patient!
    // In a browser, 'this' might be window or undefined
    return patient.medications.join(", ");
  }
};

console.log(patient.getFullName()); // "John Doe"
console.log(patient.getAge()); // 45
console.log(patient.getMedications()); // "Aspirin, Lisinopril"

Object Destructuring

Object destructuring extracts properties into variables:

  • Concise syntax for extracting multiple properties
  • Can rename variables
  • Can provide default values
  • Can destructure nested objects
  • Works in function parameters

React Native Tip: Destructuring is widely used with React props and hooks.

const medication = {
  id: 1,
  name: "Aspirin",
  dosage: "81mg",
  frequency: "once daily",
  instructions: "Take with food"
};

// Basic destructuring
const { name, dosage, instructions } = medication;
console.log(name); // "Aspirin"
console.log(instructions); // "Take with food"

// Renaming variables
const { name: medicationName, dosage: medicationDosage } = medication;
console.log(medicationName); // "Aspirin"

// Default values
const { price = "Not specified" } = medication;
console.log(price); // "Not specified"

// Nested destructuring
const prescription = {
  medication: {
    name: "Lisinopril",
    dosage: "10mg"
  },
  patient: {
    id: "P12345",
    name: "Jane Smith"
  }
};

const { 
  medication: { name: medName }, 
  patient: { name: patientName } 
} = prescription;
console.log(medName); // "Lisinopril"
console.log(patientName); // "Jane Smith"

Object Spread Operator

The spread operator for objects:

  • Create copies of objects
  • Merge multiple objects
  • Create new objects with modified properties
  • Essential for immutable state updates in React Native
const medication = {
  name: "Aspirin",
  dosage: "81mg",
  frequency: "once daily"
};

// Copy an object
const medicationCopy = { ...medication };

// Merge objects
const baseInfo = { name: "Aspirin", dosage: "325mg" };
const details = { 
  frequency: "twice daily", 
  instructions: "Take with food" 
};

const completeInfo = { ...baseInfo, ...details };
console.log(completeInfo); 
// { name: "Aspirin", dosage: "325mg", 
//   frequency: "twice daily", instructions: "Take with food" }

// Override properties
const updatedInfo = { 
  ...baseInfo, 
  dosage: "81mg" // Overrides the dosage from baseInfo
};
console.log(updatedInfo); // { name: "Aspirin", dosage: "81mg" }

// React Native state update example
function updateMedication(prevState, id, newDosage) {
  return {
    ...prevState,
    medications: prevState.medications.map(med => 
      med.id === id 
        ? { ...med, dosage: newDosage } 
        : med
    )
  };
}

Computed Property Names

Computed property names allow dynamic property keys:

  • Use square brackets [] in object literals
  • Evaluate expressions as property names
  • Useful for dynamic data structures
  • Works with variables and expressions
// Variables as property names
const propertyName = "dosageInfo";
const timeOfDay = "evening";

const medicationSchedule = {
  name: "Simvastatin",
  [propertyName]: "40mg", // Uses value of propertyName
  [`${timeOfDay}Dose`]: true // Template literal
};

console.log(medicationSchedule.dosageInfo); // "40mg"
console.log(medicationSchedule.eveningDose); // true

// Dynamic object creation with computed properties
function createMedication(name, value, unit) {
  return {
    name,
    [`${name}Dosage`]: `${value}${unit}`,
    [`${name}LastTaken`]: new Date()
  };
}

const aspirin = createMedication("Aspirin", 81, "mg");
console.log(aspirin.AspirinDosage); // "81mg"

Working with Arrays of Objects

const medicationList = [
  { id: 1, name: "Aspirin", type: "pain", dosage: "325mg" },
  { id: 2, name: "Lisinopril", type: "blood pressure", dosage: "10mg" },
  { id: 3, name: "Metformin", type: "diabetes", dosage: "500mg" }
];

// Finding a specific medication
const diabetes = medicationList.find(med => med.type === "diabetes");
console.log(diabetes.name); // "Metformin"

// Creating a new array with modified objects (immutably)
const updatedList = medicationList.map(med => {
  if (med.id === 2) {
    return { ...med, dosage: "20mg" }; // Create new object
  }
  return med; // Keep unchanged
});

// Adding a new medication (immutably)
const newMed = { 
  id: 4, 
  name: "Simvastatin", 
  type: "cholesterol", 
  dosage: "20mg" 
};
const extendedList = [...medicationList, newMed];

// Removing a medication (immutably)
const filteredList = medicationList.filter(med => med.id !== 1);

// Updating all medications with a new property
const withInstructions = medicationList.map(med => ({
  ...med,
  instructions: `Take ${med.name} as directed`
}));

Immutability in React Native

Why immutability matters in React Native:

  • React/React Native rely on reference equality checks
  • Mutating objects directly can bypass re-renders
  • Enables efficient shouldComponentUpdate
  • Makes debugging easier
  • Simplifies state management
// ❌ Incorrect way (mutating)
const updateMedicationDosage = (medications, id, newDosage) => {
  const med = medications.find(m => m.id === id);
  med.dosage = newDosage; // Mutation! Bad for React Native
  return medications; // Same array reference
};

// ✅ Correct way (immutable)
const updateMedicationDosageCorrectly = (medications, id, newDosage) => {
  return medications.map(med => {
    if (med.id === id) {
      return { ...med, dosage: newDosage }; // New object
    }
    return med;
  });
};

// In a React Native functional component
function MedicationScreen() {
  const [medications, setMedications] = useState([
    { id: 1, name: "Aspirin", dosage: "81mg" }
  ]);
  
  const updateDosage = (id, newDosage) => {
    setMedications(prevMeds => 
      updateMedicationDosageCorrectly(prevMeds, id, newDosage)
    );
  };
  
  // Component rendering...
}

Complex Data Transformations

const patientData = {
  id: "P12345",
  name: "John Doe",
  medications: [
    { id: 1, name: "Aspirin", type: "pain", dosage: "325mg", times: ["morning"] },
    { id: 2, name: "Lisinopril", type: "blood pressure", dosage: "10mg", times: ["morning"] },
    { id: 3, name: "Simvastatin", type: "cholesterol", dosage: "20mg", times: ["evening"] }
  ]
};

// Group medications by time of day
const medicationsByTime = patientData.medications.reduce((schedule, med) => {
  med.times.forEach(time => {
    if (!schedule[time]) {
      schedule[time] = [];
    }
    schedule[time].push(med);
  });
  return schedule;
}, {});

console.log(medicationsByTime.morning.length); // 2
console.log(medicationsByTime.evening.length); // 1

// Create a medication schedule
const medicationSchedule = Object.entries(medicationsByTime).map(([time, meds]) => {
  return {
    timeOfDay: time,
    medications: meds.map(med => ({
      name: med.name,
      dosage: med.dosage,
      instructions: `Take ${med.dosage} of ${med.name}`
    }))
  };
});

// Example React Native component using this data
function MedicationSchedule({ schedule }) {
  return (
    
      {schedule.map(timeSlot => (
        
          {timeSlot.timeOfDay}
          {timeSlot.medications.map(med => (
            {med.instructions}
          ))}
        
      ))}
    
  );
}
