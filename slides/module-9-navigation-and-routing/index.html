Module 9

Navigation and Routing in React Native

React Native Training Course

Module Overview

This module covers navigation and routing in React Native applications:

  • Understanding navigation fundamentals in mobile applications
  • Implementing navigation with Expo Router
  • Implementing navigation with React Navigation
  • Applying advanced navigation patterns
  • Building comprehensive navigation systems for pharmacy applications
Module Structure

4 Sections + Module Challenge | Total Time: Approximately 4-5 hours

Learning Objectives

  • Understand the fundamentals of navigation in mobile applications
  • Implement navigation using Expo Router
  • Implement navigation using React Navigation
  • Create and manage navigation stacks, tabs, and drawers
  • Handle navigation parameters and deep linking
  • Implement authentication flows
  • Apply best practices for navigation in React Native
  • Create complex navigation patterns

Module Sections

Section 1: Introduction to Navigation

Learn the fundamental concepts of navigation in mobile applications.

Section 2: Expo Router

Explore Expo Router, a file-based routing library for React Native applications.

Section 3: React Navigation

Learn how to use React Navigation, a popular navigation library for React Native applications.

Section 4: Navigation Patterns

Explore advanced navigation patterns and techniques for creating complex navigation structures.

Module Challenge

Pharmacy App Navigation

Implement a comprehensive navigation system for a pharmacy application:

  • Create various navigation patterns (stack, tab, drawer)
  • Implement authentication flows
  • Set up deep linking
  • Handle navigation state management
  • Apply proper transitions and animations

Estimated time: 60-90 minutes

  • Expo Router
  • React Navigation
  • Navigation patterns
  • Deep linking
  • Authentication flows
  • Navigation state management
  • Screen transitions and animations
