Module 9

Navigation and Routing

React Native Training Course | 2-3 hours

Module Learning Objectives

  • Understand the fundamentals of navigation in mobile applications
  • Implement navigation using Expo Router
  • Implement navigation using React Navigation
  • Create and manage navigation stacks, tabs, and drawers
  • Handle navigation parameters and deep linking
  • Implement authentication flows
  • Apply best practices for navigation in React Native

Module Journey

This module takes you on a journey from understanding basic navigation concepts to implementing complex navigation patterns in your pharmacy application. You'll learn both Expo Router and React Navigation, allowing you to choose the best approach for your specific needs.

Time Management: If running short on time, prioritize Sections 1 and 2 as they provide the foundational understanding and Expo Router, which is the recommended approach for new React Native applications. Section 3 can be covered at a higher level, focusing on the key differences between Expo Router and React Navigation.

Target Audience

This module is designed for developers with experience in:

Android Developers

  • Familiar with Android navigation components
  • Experience with Activities and Fragments
  • Understanding of Intent-based navigation

iOS Developers

  • Familiar with UINavigationController
  • Experience with Storyboards or SwiftUI navigation
  • Understanding of view controller lifecycle

Web Developers

  • Familiar with React Router or similar web routing libraries
  • Experience with single-page application navigation
  • Understanding of URL-based routing

Course Theme: Pharmacy/Medication App

In this module, we'll implement navigation for our pharmacy application:

  • Authentication flow for user login and registration
  • Tab-based navigation for main app sections
  • Stack navigation for medication details and checkout
  • Drawer navigation for additional options
  • Deep linking for sharing medication information
Navigation Requirements

Our pharmacy app requires a comprehensive navigation system that allows users to browse medications, view details, manage their cart, and track orders. The navigation should be intuitive and follow platform-specific patterns while maintaining a consistent user experience.

Module Challenge

At the end of this module, you'll complete a challenge to implement a comprehensive navigation system for a pharmacy application:

  1. Implement authentication flow with protected routes
  2. Create tab-based navigation for main app sections
  3. Implement stack navigation for detailed views
  4. Add drawer navigation for additional options
  5. Configure deep linking for sharing medication information
Challenge Preparation

Take notes throughout the module on navigation patterns and implementation details that will help you complete the challenge. Pay special attention to authentication flows and deep linking configuration.
