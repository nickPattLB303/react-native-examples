Module 1

React Native Fundamentals

React Native Training Course | 2-3 hours

Module Learning Objectives

  • Understand the mobile development landscape and where React Native fits
  • Articulate the advantages and limitations of React Native
  • Explain the core architecture of React Native
  • Navigate and utilize the React Native documentation effectively

Module Structure

Section 1: The Mobile Development Landscape

Overview of native, hybrid, and cross-platform approaches to mobile development.

Duration: 30-45 minutes

Overview: 10 min | Approaches: 15-20 min | Exercise: 5-15 min

Section 2: Why React Native?

Advantages, limitations, and ideal use cases for React Native.

Duration: 25-35 minutes

Value Prop: 10 min | Advantages: 10-15 min | Exercise: 5-10 min

Section 3: React Native Internals

Deep dive into the architecture and how React Native works under the hood.

Duration: 30-45 minutes

Architecture: 15 min | Threading: 10-15 min | Exercise: 5-15 min

Section 4: React Native Documentation

How to effectively navigate and use the official documentation.

Duration: 20-30 minutes

Resources: 10 min | Navigation: 5-10 min | Exercise: 5-10 min

Module Journey

This module takes you on a journey from understanding the broader mobile development context, to discovering why React Native stands out, exploring how it works internally, and finally learning how to find answers as you build. Each section builds on the previous one, culminating in the ability to make informed decisions about using React Native for your pharmacy application.

Time Management: If running short on time, prioritize Sections 1 and 2 as they provide the foundational understanding and value proposition. Section 3 can be covered at a higher level, focusing on the key architectural concepts rather than implementation details.

Target Audience

This module is designed for developers with experience in:

Android Developers

  • Familiar with Java/Kotlin
  • Experience with Android SDK
  • Understanding of Android UI components

iOS Developers

  • Familiar with Swift/Objective-C
  • Experience with iOS SDK
  • Understanding of UIKit components

Web Developers

  • Familiar with JavaScript/TypeScript
  • Experience with React or similar frameworks
  • Understanding of component-based architecture

Course Theme: Pharmacy/Medication App

Throughout this course, we'll be building a medication management application:

  • Track medications and dosages
  • Set reminders for taking medications
  • Record medication adherence
  • Scan prescription barcodes
  • Locate nearby pharmacies
Why a Pharmacy App?

This theme provides real-world context for learning React Native concepts while exploring common mobile app features like notifications, camera access, geolocation, and offline storage.

Module Challenge

At the end of this module, you'll complete a challenge to analyze different approaches for building a medication reminder app:

  1. Compare native, hybrid, and React Native approaches
  2. Analyze the architecture requirements
  3. Identify potential bridge communication needs
  4. Create a documentation reference guide
Challenge Preparation

Take notes throughout the module on concepts that will help you complete the challenge, especially regarding architecture considerations and platform-specific features.
