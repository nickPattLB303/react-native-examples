Exercise Instructions

In this exercise, you'll implement a set of React components for a medication tracking application:

  1. Complete the MedicationHeader component that displays a title and description
  2. Implement the StatusBadge component that displays different styles based on status
  3. Create the MedicationItem component that displays medication details and uses StatusBadge
  4. Build the MedicationList component that renders multiple MedicationItem components with proper keys
  5. Complete the MedicationApp component that brings everything together

Use the sample data provided in the starter code. Focus on applying JSX syntax, conditional rendering, and proper list rendering with keys.
