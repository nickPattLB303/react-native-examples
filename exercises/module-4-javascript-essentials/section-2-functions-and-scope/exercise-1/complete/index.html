Part 1: Function Declaration

Original function declaration:

function calculateDosage(weight, age) {
    const baseDosage = weight * 0.1;
    const ageFactor = age < 18 ? 0.8 : 1;
    return baseDosage * ageFactor;
}

Part 2: Function Expression

Function declaration converted to function expression:

const calculateDosage = function(weight, age) {
    const baseDosage = weight * 0.1;
    const ageFactor = age < 18 ? 0.8 : 1;
    return baseDosage * ageFactor;
};

Test result:

Note: Function expressions are not hoisted and must be defined before use.

Part 3: Arrow Function

Function expression converted to arrow function:

const calculateDosage = (weight, age) => {
    const baseDosage = weight * 0.1;
    const ageFactor = age < 18 ? 0.8 : 1;
    return baseDosage * ageFactor;
};

Test result:

Note: Arrow functions don't bind their own 'this' context.

Part 4: 'this' Context Test

Testing 'this' context in different function forms:

const patient = {
    name: "John",
    weight: 70,
    age: 35,
    getDosage: function() {
        return calculateDosage(this.weight, this.age);
    }
};

Note: The 'this' context is preserved in the method call because we're using a regular function. If we used an arrow function, 'this' would refer to the global scope.
