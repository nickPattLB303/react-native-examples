Pharmacy Inventory System

Exercise Instructions

Implement a TypeScript-based pharmacy inventory system using advanced types by completing the TODOs in the TypeScript file:

  1. Create a generic InventoryItem<T> interface that works with different product types
  2. Implement interfaces for different product types (medications, medical supplies)
  3. Create a type union for different inventory actions (restock, dispense, adjust)
  4. Implement type guards to handle different inventory actions appropriately
  5. Use utility types to create specialized views of your inventory data

Look for TODO comments in the TypeScript code to see what needs to be implemented.
