Medication Types System - TypeScript Challenge

This challenge focuses on creating well-defined TypeScript interfaces and types that could be used in a React Native medication tracking application.

Challenge Overview

Create a comprehensive type system for a medication tracking application. Focus only on defining the types, interfaces, and function signatures, not on implementing the actual functionality.

Note: You do not need to implement any functionality. This challenge is focused on creating a well-structured type system.

Part 1: Core Medication Types

Create a types.ts file with the core data models.

Define interfaces for:

  • Medication - Include properties like id, name, manufacturer, dosage, etc.
  • Prescription - Information about who prescribed the medication
  • DosageSchedule - When and how the medication should be taken
